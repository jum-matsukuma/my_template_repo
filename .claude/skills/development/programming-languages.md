# Programming Languages

Core programming language skills and capabilities available in this development environment.

## Frontend Technologies

### JavaScript/TypeScript
- Modern ES2022+ features and syntax
- Async/await patterns and Promise handling
- Module systems (ESM, CommonJS)
- TypeScript strict mode configuration
- Advanced type patterns and generics

### React
- Functional components with hooks (useState, useEffect, useContext, etc.)
- Custom hook patterns and best practices
- State management (Context API, Redux, Zustand)
- Performance optimization (memo, useMemo, useCallback)
- Testing with React Testing Library

### CSS/Styling
- Modern CSS features (Grid, Flexbox, Container Queries)
- CSS-in-JS solutions (styled-components, emotion)
- Preprocessors (SCSS/SASS, Less)
- CSS frameworks (Tailwind CSS, Bootstrap)
- Design systems and component libraries

- Express.js and Fastify frameworks

### Python
- FastAPI and Flask web frameworks

### Databases

- Database migrations and schema management
