# Fix Errors

エラー出力やバグ報告をもとに、原因を特定して修正する。

## Usage

- `/fix` - 直近のターミナル出力のエラーを修正
- `/fix $ARGUMENTS` - 指定されたエラー内容を修正

## What This Command Does

以下の手順で問題を解決する:

### 1. エラーの分析
- エラーメッセージ、スタックトレース、ログを解析
- 関連するソースコードを特定して読み込む
- エラーの根本原因を推測

### 2. 修正方法の決定
- 最小限の変更で問題を解決するアプローチを選択
- 副作用やリグレッションのリスクを考慮
- 複数の原因が考えられる場合は最も可能性の高いものから対処

### 3. 修正の実行
- 該当ファイルを修正
- 型チェック・リントがある場合は実行して確認
- 修正内容をユーザーに説明

### 4. 検証
- 可能であればエラーが解消されたことを確認
- テストがある場合は実行して既存機能への影響がないことを確認
