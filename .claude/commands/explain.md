# Explain Code

コードやアーキテクチャをわかりやすく説明する。

## Usage

- `/explain [file-path]` - 指定ファイルのコードを説明
- `/explain [file-path]:[line-range]` - 特定の行範囲を説明
- `/explain $ARGUMENTS` - 指定されたコード・概念を説明

## What This Command Does

以下の観点からコードを解説する:

### 1. 概要
- そのコード・モジュールの目的と役割
- 全体のアーキテクチャにおける位置づけ

### 2. 処理フローの説明
- 主要な処理の流れをステップバイステップで解説
- 重要な分岐やエラーハンドリングのポイント

### 3. 依存関係
- 使用しているライブラリ・モジュール
- 他のコードからどのように呼ばれているか

### 4. 設計上のポイント
- なぜこの実装方式が選ばれたか（推測含む）
- 注意すべき制約やトレードオフ

### Output Format

- 日本語で説明（コード用語は英語のまま）
- 必要に応じてコード断片を引用
- 図やフローが有用な場合はASCIIアートで表現
