# Development Tools

Tools and workflows available for software development in this environment.

## Build Tools and Package Managers

### JavaScript/TypeScript
- **npm/yarn/pnpm**: Package management and script execution
- **Vite**: Fast build tool and development server
- **Webpack**: Module bundling and asset processing
- **esbuild**: Extremely fast bundler and minifier
- **Rollup**: Library bundling with tree-shaking

### Python
- **pip**: Package installation and management
- **Poetry**: Dependency management and packaging
- **pipenv**: Virtual environment and dependency management
- **setuptools**: Package building and distribution

### Rust
- **Cargo**: Build system, package manager, and test runner
- **cargo-watch**: Automatic rebuilding on file changes
- **cargo-audit**: Security vulnerability scanning

## Version Control

### Git Workflows
- Feature branch workflows
- Conventional commit messages
- Pull request and code review processes
- Git hooks for automation
- Semantic versioning and release management

## Code Quality Tools

### Linting and Formatting
- **ESLint**: JavaScript/TypeScript linting
- **Prettier**: Code formatting across multiple languages
- **Ruff**: Fast Python linter and formatter
- **Clippy**: Rust linting and suggestions
- **golangci-lint**: Go static analysis

### Type Checking
- **TypeScript**: Static type checking for JavaScript
- **mypy**: Static type checking for Python
- **Rust**: Built-in type system with ownership checking

## Testing Frameworks

### Frontend Testing
- **Vitest**: Fast unit testing for Vite projects
- **Jest**: Comprehensive testing framework
- **Playwright**: Cross-browser end-to-end testing
- **Cypress**: Developer-friendly E2E testing

### Backend Testing
- **pytest**: Python testing framework with fixtures
- **Supertest**: HTTP assertion testing for Node.js
- **cargo test**: Built-in Rust testing framework
