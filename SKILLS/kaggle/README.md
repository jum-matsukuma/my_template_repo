# Kaggle Competition Skills

This directory contains specialized skills for Kaggle competition development. These skills are only loaded when working on Kaggle-related projects to avoid cluttering the context for general development work.

## Available Skills

### Setup and Tools
- `kaggle-api-setup.md` - Kaggle API installation and authentication guide

### Core Competition Skills
- `kaggle-workflow.md` - Efficient competition workflow patterns
- `competition-strategy.md` - Strategic competition management
- `ml-modeling.md` - Machine learning modeling techniques
- `data-analysis.md` - Data analysis and preprocessing

### Detailed Knowledge Base
- `kaggle-competition.md` - Competition-specific information template
- `data-understanding.md` - Dataset analysis and feature documentation
- `competition-insights.md` - Community insights and discussions
- `solution-strategy.md` - Solution approach and methodology
- `feature-engineering.md` - Feature engineering patterns
- `model-zoo.md` - Model configurations and strategies
- `evaluation-metrics.md` - Validation and evaluation frameworks

## Usage

These skills are automatically available when working in a Kaggle competition directory created from the `kaggle-template/`. For general Python development projects, these skills remain isolated to keep the development context focused.
